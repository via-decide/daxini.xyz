## 🔥 Key Features
- **Visual DAG Architecting:** Map your app logic using nodes (UI, API, DB, Auth) on a spatial matrix.
- **AI Synthesis:** Integrated Gemini 2.0 Flash engine for high-fidelity code generation.
- **Zero NPM Overhead:** Generates pure Vanilla JS, HTML, and CSS. No `node_modules` required.
- **Instant APK Compiler:** One-click generation of true compressed Android packages (.apk).
- **PWA Ready:** Installable on iOS, Android, and Desktop as a standalone application.

## 🚀 Quick Start
1. Visit [logichub.app](https://logichub.app)
2. Enter your Gemini API Key in the header.
3. Drag nodes from the sidebar to the canvas.
4. Wire nodes together to define logic flow.
5. Hit `[⚡ Synth]` to generate code and `[📱 APK]` to build your app.
